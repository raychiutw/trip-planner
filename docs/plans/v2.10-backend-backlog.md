## Item 1 — POI Search Endpoint（Wave 2）

**選 A. Nominatim proxy** — 全球資料、無 auth、免費，缺點是要 cache + rate limit。

**API**：`GET /api/pois/search?q=&near=:lat,:lng&limit=10`

```json
[{
  "name": "沖縄美ら海水族館",
  "type": "attraction",
  "address": "沖縄県国頭郡本部町石川 424",
  "lat": 26.6944, "lng": 127.8779,
  "source": "nominatim"
}]
```

**實作**：
- Server-side cache 24h（KV namespace POI_SEARCH_CACHE）
- Rate limit：1 req/sec/user（IP-based fallback）
- User-Agent header 必填（Nominatim ToS）

**前端 wire**：InlineAddPoi search input 拆掉 disabled、debounce 250ms、result list show real items + Add button → POST `/api/trips/:id/entries/:eid` 走既有 entries 端點

## Item 2 — Copy Entry Endpoint（Wave 1）

**API**：`POST /api/trips/:id/entries/:eid/copy?targetDayId=&sortOrder=`

**Body**: 可選 `{ time?, sortOrder? }` 覆寫

**Logic**：
1. perm check + verifyEntryBelongsToTrip
2. SELECT source entry
3. INSERT new entry with `day_id = targetDayId`、`sort_order` 插入到目標位置
4. SELECT trip_pois WHERE entry_id = source → INSERT 副本 with `entry_id = new`
5. logAudit action='copy' diff={source: srcEid}
6. RETURN new entry row

**前端 wire**：TimelineRail ⎘ button 開 EntryActionPopover(action='copy')，confirm → POST → dispatch `tp-entry-updated`

## Item 3 — Move Entry 跨天（Wave 1）

**選 A. PATCH `day_id` 加進 ALLOWED_FIELDS** — 1 行 code，沿用既有 PATCH 流程（perm / audit / diff 全 free）。

```diff
- const ALLOWED_FIELDS = ['sort_order', 'time', 'title', ...] as const;
+ const ALLOWED_FIELDS = ['day_id', 'sort_order', 'time', 'title', ...] as const;
```

需驗證：
- targetDayId 屬於同 trip（join trip_days where trip_id = :id）
- 不可改成不同 trip 的 day_id

**前端 wire**：TimelineRail ⇅ button 開 EntryActionPopover(action='move')，confirm → PATCH `{ day_id, sort_order }` → dispatch event

## Item 4 — POI Photos（Wave 3）

**選 B. `pois.photos` JSON column** — 1 個 migration，per-POI 共用照片，cover 80% 案例。

**Migration**：
```sql
ALTER TABLE pois ADD COLUMN photos TEXT;  -- JSON array of { url, caption?, source? }
```

**資料來源**：Wikimedia Commons API
- `GET https://commons.wikimedia.org/w/api.php?action=query&prop=imageinfo&titles=:name&iiprop=url&iiurlwidth=400`
- 寫 cron script：`scripts/populate-poi-photos.js` 每週掃 photos IS NULL 的 pois，抓 1–5 張，寫回

**前端 wire**：StopLightbox photo placeholder 替換成 carousel（◀▶ + thumbnail），讀 `entry.poi.photos`

## 風險與 mitigations

| 風險 | Mitigation |
|---|---|
| Nominatim rate limit 觸發 | Server-side KV cache 24h、單 user 1 req/sec、降級顯示 cache 結果 |
| Wikimedia API 抓不到照片 | populate-script silent fail、photos 留 NULL、StopLightbox 顯示 placeholder |
| 既有 `/stop/:eid` 分享 link 404 | StopRedirect 保 URL backward-compat，redirect 到 trip + focus |
| copy/move 觸發大量 audit log | logAudit action='copy'/'move' 額外 type，可走標準 batch flow |
