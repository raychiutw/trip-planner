隱私權政策

最後更新：[待定] 上線日期　·　適用於 Tripline 網站與行動應用程式

Tripline 是旅遊行程規劃工具。這份政策說明我們收集哪些資料、為什麼收集、傳給誰， 以及你可以怎麼取回或刪除。我們盡量寫得具體 —— 如果哪一段你看不懂實際指什麼，那是我們的問題，請告訴我們。

系統自動產生的

  • 登入憑證：密碼以 PBKDF2-HMAC-SHA256（10 萬次迭代 + 隨機 salt）雜湊儲存。我們不保存你的明文密碼。
  • 裝置與登入紀錄：瀏覽器與作業系統摘要、IP 位址的雜湊值（非原始 IP）、登入時間。
  • 操作稽核：誰在什麼時候改了哪個行程。
  • 錯誤回報：程式出錯時的網址、錯誤訊息與瀏覽器資訊。

來源：src/server/password.ts:22-41、migrations/0037（session_devices）、0036（auth_audit_log）、0071（audit_log）、0017（error_reports）

我們不收集的

  • 不追蹤你的位置軌跡。地圖的「我的位置」按鈕只在你按下時取得座標用來移動地圖畫面，不上傳、不儲存。
  • 不收集通訊錄、相簿、麥克風。
  • 不販售個人資料。

來源：src/pages/GlobalMapPage.tsx:786-796、src/components/trip/MapFabs.tsx:123-150（僅 map.panTo()，無上傳路徑）

2. 為什麼收集

  • 提供服務：儲存並同步你的行程、地圖路線、AI 規劃。
  • 帳號安全：驗證身分、偵測異常登入、讓你管理已登入裝置。
  • 共同編輯：讓你邀請的旅伴能存取指定行程。
  • 問題排查：錯誤回報與稽核紀錄用於重現與修復問題。

4. 第三方應用程式存取

Tripline 可作為登入提供者，讓第三方應用程式在你授權後存取你的資料。你可以隨時在帳號 → 已連結應用撤銷授權。

[待定 5/8] ⚠ 這段目前寫不了 —— 需先決定改 code 還是誠實揭露。
授權畫面顯示的範圍（例如「識別您的身分」）與實際能力不符： functions/api/_auth.ts:99-181 的權限判斷完全不檢查 scope， 因此任何取得授權的應用程式實際上可讀寫你全部行程（含航班編號、訂房編號、緊急聯絡人）。

曝險有限（第三方 app 須經人工核准為 active 才能走完授權流程），但照授權畫面的文字寫進本政策即為不實陳述。 兩條路：① 在權限判斷加上 scope gate；② 本段誠實寫「授權後該應用可存取您全部行程資料」。

[待定 6/8] 帳號、行程、收藏、操作稽核紀錄的保留期 —— code 中完全沒有清理邏輯， 這是政策不是程式碼，只能由你決定。常見寫法：「帳號存續期間持續保留，刪除帳號時一併移除」。
⚠ 但操作稽核（audit_log）目前沒有保留期且刪帳號時只做匿名化 —— 需一併決定是否加保留期。
[待定 7/8] 自動清理的實際執行機制。scripts/auth-cleanup.js 的註解自相矛盾 （一說由本機排程執行、一說已改雲端 cron），且 .github/workflows/ 內找不到它。 若靠單機排程，機器離線期間上表的保留期承諾都不成立，不宜寫進政策。
[待定 8/8] 資料庫備份的保留期與地理位置（Cloudflare 端設定，不在 code 內）。

查看與更正帳號頁可修改顯示名稱；行程內容可隨時編輯
取得副本每個行程可匯出為 JSON 或列印為 PDF

刪除帳號的細節

刪除後我們會移除：你的帳號與登入憑證、你建立的所有行程與其行前資料、收藏、登入階段與裝置紀錄、共編邀請紀錄。

操作稽核紀錄會匿名化保留（移除你的電子郵件與資料快照，只留「某位已刪除使用者在某時間改過某行程」的行為紀錄）， 以維持共編行程的變更歷史完整。

⚠ 你建立的行程若有共編者，那些行程也會一併刪除，共編者將失去存取權。刪除前的確認畫面會顯示受影響的人數。

補充：全帳號資料匯出（非逐行程）目前沒有。GDPR 的可攜權若要主張完整支援，需另行實作。 目前寫法只承諾「每個行程可匯出」，屬據實陳述。

7. 安全措施

  • 全站 HTTPS 傳輸加密。
  • 密碼以 PBKDF2 雜湊儲存，我們無法還原你的密碼。
  • IP 位址以雜湊值儲存，不存原始位址。
  • 內容安全政策（CSP）限制可載入的外部資源。
  • 分享連結以雜湊儲存，並設有效期。
落地前建議先修（否則上表第 3 點有瑕疵）：rate_limit_buckets 目前以 明文 IP 與明文 email 作為主鍵（migrations/0035:10-14），與「IP 以雜湊儲存」的陳述衝突。 另 SESSION_IP_HASH_SECRET 若未在正式環境設定，雜湊為無鹽 SHA-256，IPv4 可被反查。

8. 兒童隱私

本服務不針對 13 歲以下兒童設計，也不會刻意收集其個人資料。若你發現有此情況，請聯絡我們，我們會盡快移除。
