IA decisions — Variant 2
  • Sheet 從底部 88% 展開，drag handle + close × + cancel/完成 雙 footer button
  • State B：鍵盤上時 sheet 自動 shrink 到 56%（避讓 280px IME）
  • Map 在 typeahead focus 時折疊成單行（accordion 收起），suggestion pick 後展開
  • 輸入順序：title → address → map → 時間（最重要在 sheet 頂部，鍵盤上仍可見）
  • 背景 trip 頁面 dim 35% overlay，user 知道自己「半疊在 trip 上」
  • Cancel 按下 → sheet 下滑消失（無中介確認）；完成 disabled 直到 pin valid
  • 權衡：sheet 比 fullpage 保留「context」感（trip 仍可見），但 keyboard occlusion 處理需 dynamic-height（複雜度 +1）
  • Map JS API fail → sheet 內顯 error banner，仍可填 title/time/note 之後 fallback search tab
