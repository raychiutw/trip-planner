IA decisions — Variant 3
  • Two-pane layout（form 380px / map flex），map 較大 ~380px 高，pin 拖動空間大
  • 「加入景點」titlebar 一行顯示 Day + 日期 + 區域，省 day-meta sub-line
  • 右 pane 加 sidehelp 解釋為什麼地址不夠、要拖 pin（cognitive aid 不擋路）
  • 順序：title → address → 時間 → 備註（form 上下），map 在右 pane 隨時可看
  • 「已調整到正確位置」hint 移到 map pane 內，跟 pin 視覺鄰近
  • Mobile fallback：matchMedia max-width 1023px → redirect 到 fullpage route
  • 權衡：對齊既有 AddStopPage 三 tab pattern，user mental model 維持；但 desktop 需要 1024px+ 才放得下，narrow 桌面（如 13" 多視窗）會擠
  • Error：typeahead fail → silent，map fail → 整 pane 退化 banner
