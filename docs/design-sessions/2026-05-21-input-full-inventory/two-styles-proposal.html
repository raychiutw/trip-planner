Input 二系統提案 — 長值 / 短值

把目前 4 種 styles 合併為 2 種正式 system styles。 按「值的長度 + 是否固定格式」分類，每個 input 屬於其中一種。

長 tp-input-long — 一般文字輸入

用於 長文字 / 不固定格式 內容：email / 密碼 / 標題 / 地址 / 描述 / 備註 / 搜尋 / 多選下拉。
規格：padding 12 14 · border 1.5px · radius lg · bg secondary · font body (16-17px) · left align

Email登入 / 註冊
密碼登入 / 重設
行程名稱EditTripPage / NewTripPage
地址 / POI 名CustomPoiForm
搜尋POI / favorites / trips
描述 / 備註SEO / entry note
顯示語言下拉

短 tp-input-short — 固定格式短值（card + label + 大字）

用於 固定格式 / 短值：time / date / 短數字。
規格：card padding 10 14 · border 1.5px · radius md · label caption muted · value 22px bold tabular-nums
