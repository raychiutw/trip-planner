Tripline Input 全盤點 v2.33.15

截至 2026-05-21 (prod 9b8fdde)，盤點全 src/ 內 74 個 <input> / <textarea> / <select> 用法。
分 4 大層級：A 全域 base / B 共用 class / C 頁面 scoped class / D component scoped class / E hidden。
所有路徑都有 fallback：page-scoped 失效時，v2.33.12 全域 base layer 確保 Terracotta 樣式。

盤點結論

  • 74 input/textarea/select element，全部都套 Terracotta 樣式（無 native fallback）
  • v2.33.12 全域 base layer（:where() specificity 0）覆蓋所有文字類 input + textarea + select + 全 date/time + checkbox/radio accent-color
  • 14 hidden inputs（ConsentPage OAuth form）— 不需樣式
  • v2.33.9 / v2.33.15 lesson：避免 reuse 別 component 的 scoped class（ConfirmModal 的 .tp-confirm-*），單獨開時 SCOPED_STYLES 不注入會 fallback native

v2.33.x lesson — 跨 component scoped class 風險

事件版本Lesson

規則：未來新增 modal / dialog，不要 reuse 別 component 的 scoped class。 要嘛把樣式 lift 到 tokens.css 共用，要嘛 component 內自己 SCOPED_STYLES 定義。
