Sidebar Spec

Active state 是當前主功能定位，不是 breadcrumb。子頁與明細頁的 active item 依所屬主功能決定，不新增子頁 tab。

  • 寬度固定 240px（--sidebar-width-desktop）
  • 背景 --color-foreground（深棕，dark sidebar on cream page）
  • Item 高度最小 40px，內邊距 12px
  • Active background --color-accent，文字 --color-accent-foreground
  • Item 字級 14px / line-height 20px / weight 600
  • 底部 sticky 帳號區（avatar + name + email）
  • Anonymous 狀態：「帳號」入口替換成「登入」，位置不變
  • 帳號名稱嚴格限制 10 char：runtime 用 name.length > 10 ? name.slice(0, 10) + '…' : name 截字（中英混合一律按 char 計）；CSS 的 max-width + overflow: hidden 是 layout 防呆 fallback

Bottom Nav Spec

Active item 用淡底 pill + 2px top indicator + accent icon/label，明確區分於 CTA 按鈕（CTA 用實心 accent）。

  • 高度含 safe-area inset 約 72-88px
  • 背景 color-mix(--color-background 92%) + backdrop-filter: blur(14px)
  • Icon 18px，label 11px / 14px / 700
  • Active：--color-accent-subtle 底 + 2px top indicator + accent text
  • Inactive：muted text，hover 時轉 foreground
  • 頁面內容必須留出底部 padding（calc(72px + env(safe-area-inset-bottom))）
  • Hide-on-scroll：向下捲 hide、向上捲 show（DayNav 跟 bnav 共用同一套 state）
