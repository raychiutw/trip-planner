聊天頁規格摘要

  • 路由：/chat（新對話）/ /chat/:id（指定 conversation）
  • Sidebar 加段：原 5 nav 下方新增「最近對話」list（chat history） + 「+ 新對話」CTA（覆蓋原本的「新增行程」CTA）
  • Main 區結構：top bar（conv title + actions）→ conversation feed（max-width 760 居中）→ chat input wrap（sticky bottom）
  • 訊息 bubble：user 右靠齊 + accent bg；AI 左靠齊 + 白底 + border。AI bubble 內可嵌 proposal card（trip preview）+ bubble-meta footer
  • Empty state：hero greeting「想去哪裡？」+ 4 個 suggestion chip（eyebrow + text 兩行）
  • Chat input：rounded-full 56px height，左 + 附加 / 中 input / 右 mic + send（accent fill）
  • Quick actions：input 上方 4 個 outline pill bubble（+加景點 / 改交通 / 換餐廳 / 調時段），對話進行中才顯示
  • Right sheet：聊出 trip 後展開 — 跟 V2 Final 同 sheet 結構（tabs + day strip + ocean-rail），多一個「儲存到行程」按鈕（DRAFT → 正式 trip）
  • Mobile 差異：sidebar 收起、改 bottom nav；conversation feed padding 16px；input 高度降至 48px；quick actions 縮成 3 個
  • 實作對應：新建 src/pages/ChatPage.tsx 是 task §6.1 的 placeholder — mockup 提供完整 UI 規格供日後 V2-P? 階段實作
