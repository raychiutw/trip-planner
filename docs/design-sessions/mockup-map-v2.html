地圖頁規格摘要

  • 路由：/map — 全域 cross-trip 地圖；跟 per-trip /trip/:id?sheet=map 不同（後者是單 trip view）
  • Scope：view-only（不含 search、不含 add to trip）；search 屬於 /explore
  • Trip palette：每 trip 一個主色 — 沖繩 #D97848 / 首爾 #06A77D / 台中 #0EA5C7。對齊 DESIGN.md Day Palette 10 色（取前三色作 trip 區分）
  • Floating overlay（map 上層）：
    • Top-left: eyebrow card「Global Map · 所有行程 · 3 trips · 15 stops」
    • Top-right: filter chips（每 trip 一個，dot + 名稱 + stops 計數，可 toggle off）
    • Bottom-left: ▣ 全覽 / ⊕ 我的位置 pill button
    • Bottom-right: 圖層切換 + 放大/縮小
  • Marker：number circle，selected 變大 + accent fill；trip 色用 stroke 區分
  • POI popup：點 marker 浮出 — eyebrow（trip · day · stop 編號）+ title + meta + 「查看詳情 / 跳到行程」雙按鈕
  • Sheet 展開時（width 收窄至 min(420px, 32vw) — 比 trip sheet 的 780 窄，留更多 map 面積）：
  • Sheet 內容：trip dot + scope eyebrow → POI title → meta chips → 位置 / 電話 / 時段 / 距離 / 座標 → 同日其他 stop mini-list（active 高亮）
  • Sheet 主 CTA：「跳到行程」accent button（router 跳到 /trip/:id?sheet=itinerary&day=N 並 anchor 到該 stop）
  • Mobile：full-screen map（占整個 main）+ top filter chips + bottom POI card carousel（被選 trip 的 stops，左右滑換）+ bottom nav 常駐
  • 不該有的：search bar / 新增 POI / heart save — 那些都在 /explore。地圖頁只是「看你已有 trips 的地理 view」
