隱私權政策

最後更新：[待定] 上線日期

一句話版本：我們只收行程規劃需要的資料，不追蹤你的位置，不賣資料。

你可以隨時匯出行程，也可以在帳號 → 刪除帳號把一切刪光。

我們收什麼

你給的：電子郵件與顯示名稱、行程內容（地點、時間、備註）、行前資料 （航班編號、住宿與訂房編號、餐廳訂位、緊急聯絡人姓名電話）、收藏的地點、 你邀請旅伴時輸入的對方信箱、你對 AI 下的行程指令。

系統產生的：密碼的雜湊值（我們無法還原你的密碼）、登入裝置摘要與 IP 位址的雜湊（非原始 IP）、誰在何時改過哪個行程的稽核紀錄、程式出錯時的錯誤資訊。

我們不收：不追蹤位置軌跡（地圖的「我的位置」只在你按下時用來移動畫面， 不上傳不儲存）、不讀通訊錄或相簿、不賣個人資料。

為什麼收

為了讓服務能運作：儲存同步你的行程、驗證身分與偵測異常登入、讓你邀請的旅伴能共同編輯、出問題時能查明原因。

[待定 1/8] 是否納入「產品改善分析」目的 —— 取決於 Cloudflare Web Analytics 是否啟用（Dashboard 端設定，code 查無載入點）。

會傳給誰

  • Google 地圖服務：你輸入的地點搜尋文字、行程座標、起訖點（用於搜尋、地圖與車程計算）。地圖元件與網頁字型由你的瀏覽器直接載入，因此 Google 會看到你的 IP。
  • Open-Meteo：行程地點座標與日期（用於天氣）。由你的瀏覽器直接連線，對方會看到你的 IP。
  • Sentry：程式出錯時的錯誤訊息與當下的完整網址（用於錯誤監控，伺服器位於美國）。
  • Google Gmail：驗證信、密碼重設信、共編邀請信的收件者與內容。
  • Cloudflare：網站與資料庫的基礎設施。
[待定 2/8] Sentry 的資料保留期。　[待定 3/8] 寄信伺服器端的稽核紀錄內容與保留期（該程式不在本專案內）。　[待定 4/8] 管理者告警目前會把使用者與被邀請者的 email 送到 Telegram —— 建議先改 code 移除，而非在此揭露。

第三方應用程式

Tripline 可作為登入提供者。你可以隨時在帳號 → 已連結應用撤銷授權。

[待定 5/8] 授權畫面顯示的範圍與實際能力不符：權限判斷不檢查 scope，獲授權的應用實際可讀寫你全部行程。需先決定要加 scope 限制，還是在此誠實寫明。

留多久

登入稽核 30 天、裝置紀錄 30 天、共編邀請 30–90 天、錯誤回報 90 天、 API 存取紀錄 60 天、AI 健檢報告 30 天、登入 Cookie 30 天。逾期自動清除。

[待定 6/8] 帳號、行程、收藏、操作稽核的保留期 —— code 中無清理邏輯，屬政策決定。　[待定 7/8] 上述自動清除的實際執行機制需確認（若靠單機排程，機器離線期間承諾不成立）。　[待定 8/8] 資料庫備份的保留期與地理位置。

你可以做什麼

  • 取回資料：每個行程可匯出 JSON 或列印 PDF。
  • 管理裝置：帳號 → 登入裝置，可登出其他裝置。
  • 撤銷授權：帳號 → 已連結應用。
  • 刪除帳號：帳號 → 刪除帳號。會永久刪除你的帳號與你建立的所有行程，無法復原。
    ⚠ 你建立的行程若有共編者，那些行程也會一併刪除，共編者將失去存取權（確認畫面會顯示受影響人數）。 操作稽核紀錄會匿名化保留 —— 移除你的信箱與資料快照，只留「某位已刪除使用者改過某行程」的行為紀錄。

安全

全站 HTTPS；密碼以 PBKDF2 雜湊儲存，我們無法還原；IP 以雜湊儲存；分享連結有雜湊與有效期。

兒童

本服務不針對 13 歲以下兒童設計，也不刻意收集其資料。若發現請聯絡我們，我們會盡快移除。
