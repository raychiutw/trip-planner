D. A11y Checklist spec 強制要求

每項 sign-off 前必須 verify。/qa 階段 will assert via @testing-library + axe.

D1 · A11y compliance matrix role / aria / keyboard / contrast
  • Region pill + Type filter chip：wrapper role="group" + aria-label="..."，每個 chip <button aria-pressed={selected}>。不是 role="tab"（filter 不切換 panel）。
  • Selection checkbox per card：aria-label={`選取 ${row.poiName}`} 帶 row context（screen reader 不只聽到「選取」）。
  • Optimistic-delete card：父層 aria-live="polite"，狀態文字「移除中…」自動播報。
  • Loading state：main container aria-busy="true" + aria-live="polite"，一次播報而非 per skeleton。
  • Heart toggle on ExplorePage：aria-pressed={isFavorited} 表達 toggle state。
  • ConflictModal：role="dialog" aria-modal="true" aria-labelledby="conflict-title"。Focus trap + ESC 關閉。
  • Pagination nav：role="navigation" aria-label="分頁" + active page aria-current="page"。
  • 所有 interactive element 含 visible focus ring（accent ring 2-3px，DESIGN.md L561）。
  • Touch target：所有 button / link / chip min-height ≥ 44px（primary action）or ≥ 32px（secondary inline，如 card-add-link）。
  • Color contrast：foreground (#2A1F18) on background (#FFFBF5) = 14.8:1 ✓ AAA。muted (#6F5A47) on background = 5.5:1 ✓ AA。accent (#D97848) on background = 3.8:1（large text only — body text 必須在 accent bg 上用 accent-foreground 白色）。
