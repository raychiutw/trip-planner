v2.33.11 ship 摘要

新增 css/tokens.css 全域 base layer 規則保護 所有 native date/time 類 input：

  • input[type="date"|"time"|"datetime-local"|"month"|"week"]
  • color-scheme: light 鎖 light mode
  • -webkit-appearance: none 統一 cross-browser
  • background-color + color 用 Terracotta tokens
  • ::-webkit-date-and-time-value 加 color override
  • ::-webkit-calendar-picker-indicator opacity 統一

未來 native input 加入時自動 inherit 此防護，不會再有 iOS dark mode 黑底黑字事件。
EditTripPage 局部 v2.33.10 fix 保留（仍 valid，跟 global 互不衝突，更具特異性）。
