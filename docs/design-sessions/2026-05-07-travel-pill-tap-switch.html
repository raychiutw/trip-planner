實作對齊 / 邊界

  • API：PATCH /api/trips/:id/segments/:sid。Body：{ mode: 'driving'|'walking'|'transit', min?: number }。transit 必填 min（1-1440）。Response：updated segment row。
  • Backend behavior：PATCH 自動 set mode_source='user'。recompute-travel 看到 user 直接 skip。Reset 動作：PATCH with mode + mode_source='auto'（後端要支援 explicit reset，目前 PATCH 永遠寫 user — Phase γ 實作要決定是否擴 PATCH 或加 DELETE endpoint）。
  • Lock icon：用 SVG 不用 emoji（DESIGN.md inline-SVG 鐵律）。
  • Tap target：所有 mode option ≥44px（DESIGN.md HIG）。
  • A11y：dialog 用 role="dialog" aria-modal="true" aria-labelledby="…"；focus trap；Esc 關閉。Auto-focus 第一個 option（用 mounted ref）。
  • Optimistic update：Save → 先本地 update + 關 dialog，背景 PATCH。失敗 → toast + revert。Phase γ 實作可 MVP 先 await PATCH 再關 dialog（簡單可靠）。
